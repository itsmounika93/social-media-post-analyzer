# 📱 Social Media Post Analyzer using Generative AI

A Generative AI-powered application that analyzes social media posts and identifies:

- Tone
- Intent
- Communication Style
- Summary

The application uses Prompt Engineering, Groq LLM, Pydantic validation, and Streamlit UI for structured AI-powered analysis.

---

# 🚀 Features

✅ Tone Detection  
✅ Intent Classification  
✅ Communication Style Analysis  
✅ AI-generated Summary  
✅ Structured JSON Output  
✅ Pydantic Validation  
✅ Download JSON Feature  
✅ Sentiment Score Visualization  
✅ Interactive Streamlit Dashboard

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/social-media-post-analyzer.git
```

## Navigate to Project Folder

```bash
cd social-media-post-analyzer
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

# 🔑 Environment Variables

Create a `.env` file and add:

```env
GROQ_API_KEY=your_api_key_here
```

# ▶️ Run Application

```bash
python -m streamlit run app.py
```
